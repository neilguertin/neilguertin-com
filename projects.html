Footsteps

As a competitive runner, I often want to look back at and analyze my running data. Unfortunately existing platforms make this nearly impossible to do. I want to be able to answer questions like,
  • When was the last time I ran to that neighborhood?
  • What was that workout I did? I know it was at that track across town.
  • Show me all of my 5K races so I can quickly compare them.
  • What pace do I usually run my easy runs at?
  • How much am I training for this marathon compared to my last one?
Frustrated by existing platforms, I set out to build my own. It is entirely focused on look back and what you've done and where you've been (hence the name, Footsteps). It allows you to search and filter all your runs by any field: Description, Distance, Day of the week, GPS, just to name a few. It also includes some basic running analytics, calculating stress per run and also fatigue and training volume over time.
  • Data: Strava via the Strava API
  • Front end: Angular
  • Charts: Plotly
  • Back end: Flask
  • Database: SQLite
